Alanbuchanan's Pic Browse

This is a small service that allows you to search a database of images, and receive JSON output of the search.

Example usage:

https://picbrowse.herokuapp.com/api/imagesearch/Frank Sinatra // Searches for ‘Frank Sinatra’

You can also include an offset query, which can be used for pagination:

?offest=100 // Skips the first 100 search results

See what others are searching for by heading to:

https://picbrowse.herokuapp.com/api/searches
